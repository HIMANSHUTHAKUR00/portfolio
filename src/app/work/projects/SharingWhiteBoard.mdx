## Overview

It is a Pandemic Openboard Added Functionalities using Canvas & JavaScript library. This project was made to solve problems faced by students and teachers during COVID Pandemic situation. The idea here is to teach students through whiteboard but in virtual platform.

## Key Features

- **Writing Pen**: Virtual Pen available for writing.
- **Eraser**: If Someone wants to remove or erase the written content.
- **Sticky Notes**: If someone wants to add notes then sticky notes feature can help.
- ** Image Uploader**: If any user wants to upload any image on the website then this feature will help.
- ** Content Downloader**: If any user wants to download the current contents of the website then this feature will help.
- ** Redo/Undo**: For undo and redoing things, this feature will help.
-** ZoomIn/ZoomOut**: For zoom + & Zoom -.

## Technologies Used

- **Node.js**
- **JavaScript**
- **DOM**
- **CANVAS**
- ** HTML/CSS**

## Challenges and Learnings

One of the biggest challenges was ensuring that the generated code was clean and maintainable. This involved setting up intelligent mapping of code structures, as well as managing edge cases like conditional rendering. Additionally, the continuous synchronization required a robust error-handling system to prevent conflicts during development.
